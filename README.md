# Multi-Agent 管理系统

一个现代化的多智能体协同管理界面，支持拖拽排序、主Agent设置和实时对话调试。

## 功能特性

### 🤖 Agent 管理
- **拖拽排序**：支持拖拽改变Agent顺序
- **主Agent设置**：可设置任意Agent为主Agent
- **动态添加**：一键添加新的Agent
- **菜单操作**：右键菜单支持设置主Agent、删除等操作

### ⚙️ 配置选项
- **欢迎语设置**：自定义Agent欢迎消息
- **对话体验**：配置对话交互方式
- **变量与记忆**：管理对话上下文和变量
- **高级设置**：更多个性化配置选项

### 💬 对话调试
- **实时对话**：右侧对话框进行实时测试
- **消息历史**：完整的对话记录
- **输入提示**：智能的输入状态显示
- **一键清空**：快速重置对话状态

## 技术特点

- **响应式设计**：适配桌面和移动设备
- **现代UI**：简洁美观的界面设计
- **流畅交互**：丰富的动画和过渡效果
- **无依赖**：纯HTML/CSS/JavaScript实现

## 使用方法

1. 打开 `index.html` 文件
2. 在左侧管理Agent配置
3. 在右侧进行对话测试
4. 拖拽Agent卡片调整顺序
5. 点击菜单按钮进行更多操作
